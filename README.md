# Why PHP

A single-page site explaining why PHP is actually insanely good in 2026.

**Live at [whyphp.dev](https://whyphp.dev)**

## Local Development

```bash
# Install dependencies
composer setup

# Serve the application
composer dev
```

Then visit [http://127.0.0.1:8000](http://127.0.0.1:8000).
